# Office 2016 自动保存脚本

一个基于 Python + COM 的小工具，用于定时自动保存已打开的 Office 文档，支持：

- Word
- Excel
- PowerPoint

适合在 Windows 环境下，减少因忘记手动保存导致的内容丢失风险。

## 功能说明

- 定时扫描当前已打开的 Office 文档
- 仅保存有改动且可写的文件（只读文件会跳过）
- 控制台输出保存日志，便于观察运行状态
- 支持自定义保存间隔

## 运行环境

- Windows（推荐 Windows 10/11）
- Python 3.8+
- 已安装 Microsoft Office（如 Office 2016）

## 安装依赖

```bash
pip install pywin32
```

## 使用方法

在项目目录中运行：

```bash
python office_autosave.py
```

默认每 30 秒执行一次自动保存。

### 自定义保存间隔

```bash
python office_autosave.py --interval 20
```

说明：
- `--interval` 单位为秒
- 脚本限制最小间隔为 5 秒

## 日志示例

```text
[2026-03-27 10:00:00] Office 自动保存已启动，间隔 30 秒。按 Ctrl+C 停止。
[2026-03-27 10:00:30] Word 已保存: 方案文档.docx
[2026-03-27 10:00:30] 本轮完成，已保存 1 个文件。
```

## 注意事项

- 该脚本只会保存“当前已打开”的 Office 文档，不会自动打开文件
- Office 程序必须在本机运行中，脚本才能获取到对应对象
- 某些受保护文档、网络路径文档或权限受限文件可能无法保存
- 建议先在测试文档上验证效果，再用于重要工作文档

## 常见问题

### 1) 运行后一直提示“本轮无需保存”

可能原因：
- 当前没有打开任何 Word/Excel/PowerPoint 文件
- 文件没有未保存改动
- Office 进程未正常运行

### 2) 提示模块不存在（如 `win32com`）

请确认已安装依赖：

```bash
pip install pywin32
```
